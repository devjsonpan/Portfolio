🔗 **Live Site:** [jasonpandev.com](https://jasonpandev.com)

![Portfolio Screenshot](https://github.com/user-attachments/assets/9ba145e2-15c9-4eb4-b01e-99b23a2ad95f)

## ✨ Features

- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Smooth Scrolling** - Animated navigation between sections
- **Interactive UI** - Hover effects, fade-in animations, active link highlighting
- **Project Showcase** - Displays my top projects with tags and links
- **Experience Timeline** - Visual representation of my journey
- **Contact Section** - Easy ways to reach me

## 🛠️ Built With

- **HTML5** - Semantic structure
- **CSS3** - Custom styling with CSS Grid and Flexbox
- **JavaScript** - Smooth scroll, animations, and interactivity
- **Font Awesome** - Icons for visual enhancement
- **Google Fonts** - Inter font family

## 🚀 Running Locally

1. **Clone the repository**
   
   ```bash
   git clone https://github.com/devjsonpan/Portfolio.git
   cd Portfolio
   ```

2. **Open in browser**
   
   Simply open `index.html` in your browser.
